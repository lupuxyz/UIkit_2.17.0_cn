可嵌套/Nestable

创建可以通过拖拽排序的可嵌套式列表。

可嵌套组件允许你通过拖拽排序条目。这是非常有用的，比如在管理界面中希望组织不同的分类或者菜单条目时

用法

嵌套列表包括列表的本身和其内容条目，手柄和拨动器。

class和data属性 描述
.uk-nestable 把这个class放到 <ul> 中定义该组件.
data-uk-nestable 要使必要的JavaScript生效，添加这个data属性即可。
.uk-nestable-item 把这个class放入每个列表条目内部的 <div> 标签上.
.uk-nestable-handle 这个class是创建手柄，使拖拽排序成为可能。
data-nestable-action="toggle" 这个data属性允许你去隐藏或显示嵌套列表条目.

        <div class="uk-nestable-item">

        </div>
    </li>
</ul>

多个列表的排序

要实现条目的跨列表排序，你可以通过对每个列表添加 data-uk-nestable="{group:'GROUP-NAME'}" 属性将多个列表划为一组。在默认情况下，不含任何组名的多个列表自动地处于同一个组中。

禁用嵌套

禁用列表条目的嵌套，你只需添加 data-uk-nestable="{maxDepth:1}" 属性。当然你还可以使用此属性来确定要嵌套的深度能有多大。

  • Item 1
  • Item 2
  • Item 3

Markup

<ul class="uk-nestable" data-uk-nestable="{maxDepth:1}">...</ul>

JavaScript 选项

这是一个如何使用data属性来设置选项的例子:

data-uk-nestable="{maxDepth:0, group:'widgets'}"
选项 可用值 默认 描述
group integer (0..6) 0 列表组
maxDepth integer 10 最大可嵌套层数

手动初始化元素

var nestable = UIkit.nestable(element, { /* options */ });

事件

名称 参数 描述
start.uk.nestable event, nestable object 拖拽开始时触发
move.uk.nestable event, nestable object 移动可嵌套条目时触发
stop.uk.nestable event, nestable object 拖拽终止时触发
change.uk.nestable event, nestable item, action 改变可嵌套条目时触发
