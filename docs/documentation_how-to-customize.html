如何自定义

使用定制工具创建你自己的风格样式。

UIKit 配备了一个 定制工具 ，你只需点击几下鼠标就可以对主题做出相应的调整，这不需要任何的CSS知识。你可以下载你全新的CSS文件，甚至是LESS的变量文件，这些都可以立即投入使用。

注意 为了达到最佳性能，我们建议禁用加载项，比如 Firefox 中的Web开发者工具 Firebug。

用法

在选择框中选择你希望自定义的主题。点击操作框中的彩色方块打开对话框修改颜色。点击文本框并输入一个数值来调整尺寸，比如width（宽度）和margin（边距）。你甚至可以使用不同的单位，定制器会自动的进行重新计算。如果对调整的结果感到满意之后，点击 Get CSS 来下载这个主题的CSS文件，并把它复制到你的 UIkit /css 文件夹中。

高级模式

定制器中的变量被分成了两个部分，默认状态下显示的变量和只出现在 高级模式 中显示的变量。默认可见的变量通常是全局变量，因为它们通常被用来定义组件变量的数值。要查看组件的变量，只需选中 “高级模式” 。

More

默认情况下，变量的值是通过另一个被隐藏变量来进行定义。在高级模式中你可以看到一些后面带有 (more) 按钮的组，这里面便是被那些影藏的变量。这些选项功能非常强大，可以自定义任意组件的变量值。

压缩 CSS

在下载你的主题前，选中 压缩 CSS ， UIkit将会删除CSS文件中所有的换行符和不必要的空格，从而大大减少文件的体积。

RTL 模式

UIKit拥有“从右向左”排列的语言本土化选项。所有你需要做的就是点击 RTL 模式，然后整个主题会自动转换。

下载 LESS/Get Less

你也可以选择下载一个只有变量的LESS文件，并在你的项目中使用。
